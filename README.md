# clipx

**clipx** — портативный менеджер буфера обмена, написанный на языке **Vala** с использованием библиотеки **GTK 3**. Программа ориентирована на работу в окружении **MATE** и других GTK-системах.

### Функционал
* **История и Избранное**: хранение последних записей буфера с возможностью их закрепления.
* **Настройки**: управление лимитом записей, очистка данных и переключение видимости панели управления.
* **Системная интеграция**: работа через системный трей, управление автозагрузкой и отображение в меню приложений.
* **Автономность**: автоматическое создание базы SQLite и файлов конфигурации в `~/.config/clipx` при первом запуске.

### Зависимости
Для работы программы требуются следующие системные компоненты:

| Компонент | Пакет в APT |
| :--- | :--- |
| **GTK+ 3** | `libgtk-3-0t64` (или `libgtk-3-0`) |
| **SQLite 3** | `libsqlite3-0` |
| **GLib 2.0** | `libglib2.0-0t64` |

**Команда для быстрой установки:**
```bash
sudo apt update && sudo apt install libgtk-3-0t64 libsqlite3-0 libglib2.0-0t64
```
